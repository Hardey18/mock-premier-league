## MOCK PREMIER LEAGUE

#### Starting the app locally
`Run the following commands:`
```
To install dependencies `yarn`
To compile `yarn compile`
To start the app `yarn dev`
To run tests `yarn test`
```
#### Starting the app with docker
```
docker-compose up --build
```
